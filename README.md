## Objective
Description
This project is based on SQL fundamentals and focuses on managing and analyzing student performance data. A Students database is created to store student details such as name, gender, age, grade, and subject-wise scores. Various SQL queries are used to perform data analysis and updates.

## Files
- `internship Assignment1 Data analysis with sql.sql`  
 

## Tasks Performed
The following tasks are completed in this project:

1. Created a database and a Students table.
2. Inserted sample data for multiple students.
3. Displayed all student records to get an overview of the data.
4. Calculated the average scores for each subject.
5. Identified the student(s) with the highest total score.
6. Counted the number of students in each grade.
7. Calculated the average score for male and female students.
8. Identified students who scored more than 80 in Mathematics.
9. Updated the grade of a student using a specific Student ID.


## Project Outcome
This project helps in understanding SQL operations such as data insertion, selection, filtering, aggregation, grouping, and updating records. It also improves skills in analyzing data and drawing meaningful insights from databases.
